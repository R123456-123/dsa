-> Bucket Sort is a distribution sorting algorithm that distributes elements into several groups (called buckets).
 Each bucket is then sorted individually (using another sorting algorithm like insertion sort, merge sort, or quicksort).
 Finally, all sorted buckets are concatenated to get the final sorted array. 

 function bucketSort(arr):
    n = length(arr)
    create n empty buckets

    for each element x in arr:
        index = int(n * x)   // bucket index
        insert x into bucket[index]

    for each bucket:
        sort(bucket)  // insertion sort or other

    return concatenate(buckets)
